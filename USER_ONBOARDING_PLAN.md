# USER ONBOARDING FLOW - STREAMLINED PLAN

## Overview
Post-signup onboarding flow taking inspiration from the split-screen login/signup concept. Collecting essential contextual user data in 8 sequential phases.

---

## ONBOARDING PHASES (8)

### PHASE 1: Welcome & Orientation
**Screen:** Welcome / Getting Started
**Goal:** Warm welcome, set expectations, establish visual continuity
**Elements:**
- Split-screen layout (left brand imagery, right content)
- Animated brand intro sequence
- "Welcome to your precision workbench" message
- Progress indicator (1/8 phases)
- Primary CTA: "Let's Get Started"
- Skip option for power users
**Data Collected:** None

---

### PHASE 2: Account Type Confirmation
**Screen:** Choose Your Path
**Goal:** Confirm user type for tailored experience
**Elements:**
- Split cards with icons for each user type:
  - Individual (Personal Fit Profile)
  - Artisan (Measurement Professional)
  - Merchant (Business Owner)
  - Enterprise (Large Organization)
- Visual representation of each persona
- Current selection highlight with mint accent
- Tooltips explaining each type
- Progress: 2/8
**Data Collected:** account_type (individual|artisan|merchant|enterprise)

---

### PHASE 3: Geographic Context
**Screen:** Your Location
**Goal:** Collect country for regional settings, currency, compliance
**Elements:**
- Searchable country dropdown with flags
- Auto-detected country pre-selected
- Region/state selection (if applicable)
- Currency auto-display for region
- Progress: 3/8
**Data Collected:** country, region, currency

---

### PHASE 4: Business Profile
**Screen:** Business Details
**Goal:** Basic business intelligence for merchants/artisans
**Elements:**
- Business/Company name
- Business type (retail|wholesale|manufacturing)
- Business address (simplified)
- Only shown for Merchant/Artisan/Enterprise account types
- Progress: 4/8 (skip for Individual)
**Data Collected:** company_name, business_type, business_address

---

### PHASE 5: Industry Context
**Screen:** Your Industry
**Goal:** Understand vertical for tailored experience
**Elements:**
- Industry vertical selector with icons and regional context
- Multi-select sub-specialties based on vertical selection
- Progress: 5/8 (skip for Individual)
**Data Collected:** industry, sub_specialties

---

### PHASE 6: Measurement Preferences
**Screen:** Measurement Standards
**Goal:** Set default units and measurement system
**Elements:**
- Unit preference: Metric (cm) / Imperial (inches)
- Toggle switch with visual preview
- Measurement standard selector:
  - Standard body measurements
  - Alvanon sizing
  - Custom measurement set
- Progress: 6/8
**Data Collected:** unit_preference (metric|imperial), measurement_standard

---

### PHASE 7: Subscription Tier
**Screen:** Plan Selection
**Goal:** Select subscription tier
**Elements:**
- Tier comparison cards:
  - **Starter** - Free (1-50 scans/mo)
  - **Pro** - $25/mo (unlimited scans)
  - **Business** - $99/mo (team + API)
  - **Enterprise** - Custom (full platform)
- Current pre-selected based on account type
- Feature comparison matrix
- "Contact Sales" for Enterprise
- Progress: 7/8
**Data Collected:** selected_plan (starter|pro|business|enterprise)

---

### PHASE 8: Completion & Success
**Screen:** Workspace Ready
**Goal:** Success confirmation, redirect to dashboard
**Elements:**
- Celebratory success animation
- Summary of configured items:
  - Account type
  - Location
  - Industry vertical (if applicable)
  - Plan
- "Go to Dashboard" CTA
- Option to "Complete Profile Later" (links to settings)
- Progress: 8/8
**Data Collected:** onboarding_completed (timestamp)

## LAYOUT DESIGN

- **Left Panel (Brand/Visuals):** Animated gradient background with contextual imagery based on phase
- **Right Panel (Interactive Content):** Phase title, form elements, navigation controls, progress indicator

### Mobile Adaptation
- Sequential full-screen cards
- Bottom navigation dots
- Swipe gestures for navigation
- Header image instead of left panel

## TECHNICAL SPECIFICATIONS

### Data Persistence
- Local storage for draft progress
- Auto-save on each phase completion
- Resume from last incomplete phase
- Direct link to specific phase: `onboarding?phase=N`

### Conditional Logic
- **Phases 4-5:** Only shown for Merchant/Artisan/Enterprise account types
- **Individual:** Skips to Phase 6 (Measurement Preferences)

### Skip Logic
- "Skip for now" available on Business Profile and Industry Context phases
- Required fields: Account type (Phase 2), Country (Phase 3), Plan (Phase 7)

### Post-Onboarding Configuration
Remaining settings available in Profile Settings:
- Avatar upload
- API key management
- Widget customization
- Team invitations
- Project setup
- Notification preferences
- Data export & privacy settings
- Integration configuration
