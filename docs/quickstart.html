Quick Start Guide

Get up and running with AI Body Scan API in under 5 minutes.

1. Get Your API Key

Sign up at https://aiscan.io/dashboard to get your API key.

YOUR_API_KEY

2. Make Your First Request

Extract measurements using dual photos (recommended for ±1-3cm accuracy):

# cURL example - Dual photo mode (recommended)
curl -X POST "https://api.aiscan.io/v2/measurements/extract" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "front=@front.jpg" \
  -F "side=@side.jpg" \
  -F "height=175" \
  -F "gender=male"

3. Response

You'll receive detailed body measurements:

{
  "success": true,
  "measurements": {
    "Shoulder": 46.4,
    "Chest Round": 102.9,
    "Waist Round": 82.4,
    "Hip Round": 97.8,
    "Inseam": 80.3,
    "Full Top Length": 77.2
  },
  "image_id": "img_1234567890",
  "gender": "male",
  "accuracy_mode": "dual",
  "accuracy": "±1-3cm",
  "processing_time_seconds": 1.8
}

Photo Requirements

📷 Front Photo

Stand straight, arms at sides, full body visible. Light in front.

📷 Side Profile

Turn 90° from camera, arms at sides, full body visible.

📏 Scale Reference

Include a reference object (e.g., credit card, A4 paper) for improved accuracy. Without reference, provide your exact height in cm.
